# auto-memory Skill

> **作用**：自动化记忆管理 - 读取、写入、整合
> **触发**：会话启动 / 用户消息 / Cron定时
> **版本**：v1.0

---

## 功能1：启动时记忆加载

当 Agent 启动时，自动执行：

```
1. 读取 IDENTITY.md    → 我是谁
2. 读取 USER.md        → 用户是谁
3. 读取 MEMORY.md      → 团队是干什么的、有哪些项目
4. 读取 memory/昨天.md → 昨天干了什么
5. 读取 memory/今天.md → 今天已经干了什么
6. 读取 projects/      → 各项目状态
7. 执行 Mem0 搜索      → 相关历史记忆
```

### 启动记忆加载 Prompt

```markdown
## 启动记忆加载

你启动时，必须先加载以下记忆：

### 1. 我是谁
读取 IDENTITY.md，获取我的身份设定

### 2. 用户是谁
读取 USER.md，获取用户信息（姓名、家人、偏好）

### 3. 团队使命
读取 MEMORY.md，了解：
- 团队定位（App开发团队）
- 团队架构（CEO+Product+Developer+Tester）
- 当前项目（说不/Budolist）

### 4. 昨天干了什么
如果 memory/昨天.md 存在，读取并总结：
- 完成的工作
- 重要决策
- 遇到的问题

### 5. 今天要干什么
如果 memory/今天.md 存在，读取：
- 待办任务
- 进行中的项目
- 今日计划

### 6. 项目状态
读取 projects/ 目录下的项目状态文件

### 7. Mem0 搜索
执行 memory_search，搜索与当前任务相关的历史记忆

---
输出格式：
"📖 记忆加载完成

✅ 我是：周小墨（CEO Agent）
✅ 用户：周涛（周总）
✅ 团队：App开发团队（Flutter + Supabase）
✅ 项目：说不(Budolist) v2.2.1
📅 昨天：完成了多Agent架构迁移
📋 今天：等待用户第一个任务"
```

## 功能2：自动记忆写入

当用户消息中出现以下关键词时，自动写入对应文件：

| 关键词 | 写入文件 | 内容 |
|--------|---------|------|
| 老婆/老公/女儿/儿子/爸妈/家人 | USER.md | 家人信息 |
| 喜欢/讨厌/想要/希望/偏好/习惯 | USER.md | 个人偏好 |
| 决定/就这样/用这个/选择 | MEMORY.md | 重要决策 |
| 删了/更新了/修改了/配置了/新增了 | memory/今天.md | 操作记录 |
| 完成了/解决了/排查到 | memory/今天.md | 任务完成 |
| 项目完成/项目失败/项目启动 | MEMORY.md | 项目记录 |
| 以后都/规则是/流程是 | RULES.md | 团队规范 |

### 自动写入 Prompt

```markdown
## 自动记忆写入

检测到以下需要记忆的内容，自动写入：

### 写入目标分析
[分析用户消息，提取需要写入的内容]

### 执行写入
[将内容追加到对应文件]

### 写入确认
"💾 已自动记忆：
- 📁 USER.md：记录了 [家人信息/偏好]
- 📁 MEMORY.md：记录了 [决策/项目]
- 📁 memory/今天.md：记录了 [操作/任务]"
```

## 功能3：每日记忆整合

定时（Cron）触发，整合记忆：

```
1. 收集今日所有记忆
2. 提取：完成的任务、决策、问题、用户反馈
3. 更新长期记忆（MEMORY.md）
4. 清理过期的临时记忆
5. 生成明日待办
```

### 整合 Prompt

```markdown
## 每日记忆整合

执行每日记忆整合任务：

### 1. 收集今日记忆
读取 memory/今天.md

### 2. 提取关键信息
- 完成的任务：[]
- 重要决策：[]
- 遇到的问题：[]
- 用户反馈：[]

### 3. 更新长期记忆
将关键信息追加到 MEMORY.md

### 4. 生成明日待办
基于今日未完成事项 + 长期记忆中的待办

### 5. 清理
- 标记今日记忆为"已整合"
- 移动到 memory/history/ 目录
```

## 技术实现

### 文件结构

```
~/.openclaw/workspace/
├── memory/
│   ├── YYYY-MM-DD.md      # 每日记忆（当日使用）
│   ├── today.md           # 今日记忆（快捷方式）
│   ├── yesterday.md       # 昨日记忆（快捷方式）
│   └── history/           # 历史记忆归档
│       ├── 2026-04-01.md
│       └── ...
├── MEMORY.md              # 长期记忆
├── USER.md                # 用户信息
└── auto-memory skill      # 本技能
```

### 快捷方式更新

每次启动时，自动更新：
- `memory/today.md` → 指向 `memory/YYYY-MM-DD.md`
- `memory/yesterday.md` → 指向昨天的文件

### 触发机制

| 触发 | 执行 |
|------|------|
| 会话启动 | 功能1：启动记忆加载 |
| 用户消息 | 功能2：自动记忆写入（异步） |
| Cron 21:00 | 功能3：每日整合 |
| Cron 周五18:00 | 功能3：每周整合 |
