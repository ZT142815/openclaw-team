# HEARTBEAT.md - Developer Agent 心跳任务

> **作用**：定义周期性自动执行的任务
> **执行者**：Developer Agent
> **版本**：v2.0

---

## 心跳任务配置

```markdown
# Developer 心跳任务：每30分钟检查一次

## 任务1：检查进行中的开发
- 触发条件：每30分钟
- 执行内容：
  1. 检查当前是否有进行中的开发任务
  2. 检查代码是否有未提交的更改
  3. 如有待处理任务，记录

## 任务2：检查测试状态
- 触发条件：每30分钟
- 执行内容：
  1. 检查是否有待修复的Bug
  2. 检查测试覆盖率
  3. 如有问题，记录
```

---

## 注意事项

- 心跳任务仅检查和记录，不执行具体工作
- 如需执行具体任务，通过 CEO 派发
- 遇到紧急问题，报告 CEO
