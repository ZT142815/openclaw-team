## 使用说明

### 何时需要复盘

| 级别 | 是否需要复盘 | 复盘时间 |
|------|-------------|---------|
| P0 | ✅ 必须 | 24小时内完成初稿 |
| P1 | ✅ 必须 | 48小时内完成初稿 |
| P2 | ⚠️ 建议 | 1周内完成 |
| P3 | ❌ 可选 | 可选择性复盘 |

### 复盘流程

1. **收集信息** - 收集日志、监控数据、时间线
2. **初步分析** - 确定直接原因和根本原因
3. **制定措施** - 提出具体的改进措施
4. **编写文档** - 填写复盘模板
5. **评审通过** - 相关人员评审
6. **跟踪执行** - 跟踪行动项完成情况

### 复盘原则

- **非责怪** - 复盘是为了改进，不是追责
- **事实导向** - 基于数据和事实
- **可操作** - 措施必须具体可执行
- **可跟踪** - 所有行动项必须有负责人和截止日期
